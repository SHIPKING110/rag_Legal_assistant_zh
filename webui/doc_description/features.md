# 功能介绍

## 核心能力
1. **法律问答**：针对中华人民共和国法律问题进行问答，结合知识库和大模型给出参考答案。
2. **相关性判断**：智能识别问题是否与法律相关，非法律问题直接转为闲聊模式。
3. **检索增强生成（RAG）**：结合向量检索、Hybrid 检索以及 Rank 重排序，提升答案准确度。
4. **回退机制**：当主模型不可用时，可自动切换备用模型或提供临时摘要，保障可用性。

## 使用说明
- 左侧功能模块中可配置模型、参数、检索策略等。
- 对话过程中会展示参考条文与思考过程，便于追溯。
- 文档说明模块可在不同文档之间切换，查看功能说明、模型详情以及 API 申请方式。
